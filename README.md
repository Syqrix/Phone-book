# Phone book app
## About the project
I developed this project to practice my skills. It is a simple console-based phone book application, similar to what you might find on your phone. Features:
+ Create, read, update, delete contacts (CRUD operations)
+ Saving data to a JSON file
+ Data validation
## Tools
+ Python: 3.12.3
+ Packer manager: Uv
+ Utilites: MyPy, Ruff, Pytest, Black
## Instalation
How to install:

1) git clone https://github.com/Syqrix/Phone-book.git
2) cd phone-book
3) In your IDE in console: "uv/pip install -r requirements.txt"
4) Write down in your console: python/python3 main.py
